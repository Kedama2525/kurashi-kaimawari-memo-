# サイト設計メモ

## コンセプト

暮らしの買い回りメモは、楽天の商品リンク集ではなく、楽天で生活用品を買う前に見る買い回りガイド型メディアにする。

```text
楽天で日用品・食品・キッチン用品を買う人向けの
商品比較・買い回り・楽天経済圏ガイド
```

## 内部リンクの流れ

```text
トップページ
↓
カテゴリページ
↓
買い回り攻略記事
↓
商品まとめ記事
↓
楽天商品リンク
```

将来的には、商品まとめ記事から楽天カード・楽天経済圏記事にもつなぐ。
